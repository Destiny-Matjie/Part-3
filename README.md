## Project Overview

This project is a responsive multi-page website developed for the Gift of the Givers Foundation. The website was created as part of the WEDE5020 Portfolio of Evidence and provides visitors with information about the organisation, its humanitarian services, and ways to contact or support the foundation.

The website uses semantic HTML5, external CSS, JavaScript, responsive design principles, Search Engine Optimisation, and accessibility considerations.
